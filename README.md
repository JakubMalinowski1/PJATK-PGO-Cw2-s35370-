# Projekt: Mini-Biblioteka

## Opis projektu
[cite_start]Projekt został zrealizowany w ramach ćwiczeń z przedmiotu *PGO* (Podstawy Programowania Obiektowego)[cite: 2, 6]. [cite_start]Celem aplikacji jest symulacja prostego systemu zarządzania zasobami bibliotecznymi w konsoli[cite: 8, 17]. [cite_start]Program pozwala na dodawanie książek do zasobów, zarządzanie danymi czytelników oraz przeprowadzanie operacji wypożyczania i zwracania pozycji[cite: 23, 31, 38, 48].

[cite_start]Całość została podzielona na etapy, z których każdy kończył się osobnym zatwierdzeniem zmian w repozytorium (commit)[cite: 9, 20].

## Lista klas i ich funkcjonalności
[cite_start]W ramach projektu powstały następujące klasy[cite: 22, 59]:

* **Ksiazka**: Reprezentuje pojedynczą pozycję w bibliotece. [cite_start]Przechowuje informacje o tytule, autorze, liczbie stron oraz aktualnej dostępności[cite: 25, 29].
* **Czytelnik**: Opisuje użytkownika biblioteki. [cite_start]Zawiera dane osobowe, numer karty oraz licznik aktualnie wypożyczonych książek[cite: 32, 33].
* **Biblioteka**: Klasa zarządzająca kolekcją obiektów. [cite_start]Wykorzystuje tablicę do przechowywania książek i zawiera logikę wyszukiwania oraz obsługi wypożyczeń[cite: 38, 39, 40].
* [cite_start]**Main**: Główna klasa programu, w której zaimplementowano scenariusz testowy sprawdzający poprawność działania wszystkich metod[cite: 28, 49, 53].
