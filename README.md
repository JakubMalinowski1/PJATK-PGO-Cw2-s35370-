# Projekt: Mini-Biblioteka

## Opis projektu
Projekt został zrealizowany w ramach ćwiczeń z przedmiotu *PGO* (Podstawy Programowania Obiektowego). Celem aplikacji jest symulacja prostego systemu zarządzania zasobami bibliotecznymi w konsoli[cite: 8, 17]. Program pozwala na dodawanie książek do zasobów, zarządzanie danymi czytelników oraz przeprowadzanie operacji wypożyczania i zwracania pozycji.

Całość została podzielona na etapy, z których każdy kończył się osobnym zatwierdzeniem zmian w repozytorium.

## Lista klas i ich funkcjonalności
W ramach projektu powstały następujące klasy:

* **Ksiazka**: Reprezentuje pojedynczą pozycję w bibliotece. Przechowuje informacje o tytule, autorze, liczbie stron oraz aktualnej dostępności.
* **Czytelnik**: Opisuje użytkownika biblioteki. Zawiera dane osobowe, numer karty oraz licznik aktualnie wypożyczonych książek.
* **Biblioteka**: Klasa zarządzająca kolekcją obiektów. Wykorzystuje tablicę do przechowywania książek i zawiera logikę wyszukiwania oraz obsługi wypożyczeń.
* **Main**: Główna klasa programu, w której zaimplementowano scenariusz testowy sprawdzający poprawność działania wszystkich metod.

## Wykorzystane technologie
* Język Java
* System kontroli wersji Git (GitHub)
* Programowanie obiektowe (klasy, pola, metody, konstruktory)
